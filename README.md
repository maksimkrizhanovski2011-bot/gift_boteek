# 🎁 Gift Stars Bot

Telegram-бот для продажи подарков за Telegram Stars (⭐)

---

## 📁 Структура файлов

```
gift_bot/
├── bot.py              ← Запуск бота
├── config.py           ← Токен и настройки (ЗАПОЛНИ!)
├── database.py         ← База данных SQLite
├── keyboards.py        ← Все кнопки
├── requirements.txt    ← Зависимости
└── handlers/
    ├── user.py         ← Функционал пользователей
    ├── admin.py        ← Админ-панель
    └── payment.py      ← Оплата Stars
```

---

## ⚙️ Установка и запуск

### 1. Получи токен бота
- Открой @BotFather в Telegram
- /newbot → получи токен
- **Важно:** включи платежи Stars: /mybots → выбери бота → Payments → Stars

### 2. Настрой config.py
```python
BOT_TOKEN = "1234567890:AAF..."   # твой токен
ADMIN_IDS = [123456789]           # твой Telegram ID (узнай у @userinfobot)
SHOP_NAME = "🎁 Gift Shop"        # название магазина
```

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Запусти
```bash
python bot.py
```

## ⭐ Как работают Stars

- Telegram Stars (XTR) — официальная валюта Telegram
- Пользователь нажимает «Купить» → открывается окно оплаты Telegram
- После успешной оплаты бот автоматически фиксирует покупку
- Для включения Stars: @BotFather → /mybots → Payments → Stars ✅

## 📝 Добавление подарка (через бота)

1. `/admin` → 📦 Товары → нажми кнопку «➕ Добавить подарок»
2. Введи: название → описание → цена в Stars → эмодзи → остаток → фото
3. Готово! Подарок появится в каталоге.
